SNCF Project: Cheapest Path Finder

API Optimization Graph Search Python C++ Simulation

Overview

This project involved developing an intelligent algorithm to compute the cheapest path between two railway stations on a map. The project was completed using SNCF Open Data and involved parsing station and route data to create a navigable graph. The algorithm was implemented in C++, utilizing Dijkstra’s algorithms with early termination for efficient pathfinding.

Key Features

  • Graph of stations and routes parsed from map data
  • Dijkstra’s algorithms with early termination for path planning (in C++ for efficiency and memory management)
  • FastAPI API and Streamlit frontend and frontend separation

Technologies Used

  • C++: Core logic and algorithm design, API
  • HTML/CSS/JS: For frontend
  • Python: API for some high level functions

Outcome

The resulting app is capable of dynamically computing optimal paths (in terms of price) through a network of stations, with potential applications in ticket sales.

Next Steps

Future improvements include designing new algorithms to further improve the computation complexity and memory complexity.
